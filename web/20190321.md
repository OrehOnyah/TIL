# 강의 PDF 내용 정리
## `<table>`
``` html
<table style="width:100%">
  <tr>
    <th>Firstname</th>
    <th>Lastname</th> 
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>80</td>
  </tr>
</table>
```
### table style 넣기 #1
``` html

<style>
table, th, td {
  border: 1px solid black;
  border-collapse: collapse; //선이 한 개로 합쳐진다.
}
th, td {
  padding: 15px; //표의 각 셀 안에 padding이 적용된다.
}
th {
  text-align: right; //표의 맨 위 한 줄(제목 줄)이 우측정렬된다.
}
</style>
```
